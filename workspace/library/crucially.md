Inside, a miniature, bioluminescent fungus – *Mycena luxaeterna* (carefully cultivated for its predictable glow cycle) is suspended in a nutrient-rich gel. A built-in timer – set by the user – dictates how long the bloom remains active, starting with just 5 minutes. The longer the task is worked on, the more the fungus's bioluminescence shifts towards warmer, richer hues, creating a rewarding visual cue.
